# NFTme - Premium NFT Marketplace

NFTme is a premium, responsive NFT marketplace landing page designed to showcase digital art and collections. It features a modern, glassmorphic UI with vibrant aesthetics and smooth interactions.

## Features

-   **Modern Design**: Utilizes glassmorphism, dynamic gradients, and a dark theme for a premium feel.
-   **Responsive Layout**: Fully responsive design that works seamlessly across desktop, tablet, and mobile devices.
-   **Interactive Elements**: Hover effects, 3D card tilts, and smooth transitions enhance user engagement.
-   **Sections**:
    -   **Hero**: Captivating introduction with a 3D tilted card visual and stats.
    -   **Featured Art**: Highlighted artwork with auction timer.
    -   **Trending**: Masonry-style grid showcasing trending NFTs.
    -   **Popular Artists**: Horizontal scroll/grid of top artists.
    -   **Community**: Call to action to join the platform.
    -   **Footer**: Comprehensive footer with social links and navigation.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling with variables, flexbox, grid, and animations.
-   **JavaScript (Vanilla)**: For mobile menu toggling and potential future interactivity.
-   **Google Fonts**: "Outfit" font family for clean typography.

## Setup & usage

This is a static web project. You can view it by simply opening `index.html` in your browser.

For a better development experience (hot reloading), you can use a simple server like standard `http-server` or VS Code's "Live Server" extension.

No compilation or build steps are required.
